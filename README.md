#ECOPERCEPT
 Eco-Friendly Consumer Purchase Prediction

 Project Overview
This project analyzes consumer perceptions and predicts eco-friendly purchase behavior using BERT embeddings and CatBoost.

 Technologies
- Python
- BERT
- Sentence Transformers
- CatBoost
- Scikit-learn

 Workflow
1. Data preprocessing
2. Text embedding using BERT
3. Feature engineering
4. Hybrid ML model training
5. Evaluation and prediction

 Model
Hybrid BERT + CatBoost classifier.

 Accuracy
80.2%
